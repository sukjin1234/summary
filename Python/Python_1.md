## Python 정리
```
** python의 변수는 데이터 값을 저장하는 것이 아닌 데이터 값의 주소값을 저장한다. **
```
```
input() : 입력 값을 받음, 입력값은 모두 문자열
print() : ()안에 내용을 모니터에 출력, 자동 줄바꿈, 출력 타입은 문자열 
    print("", end="") -> 줄바꿈 X
```
### 자료형
```
정수 
    생성자 : int()
    정해진 크기 제한 없음

실수
    생성자 : float()
    8비트

논리형                      type() : 자료형 종류 확인인
    생성자 : bool()
    단 2개의 값만 갖는 자료형 : True(참), False(거짓)
```
### 변수
```
변수 : (파이썬에서는) 객체(자료, 데이터, 값)를 가리키는 것
    ex) a = 1
        메모리에 1을 저장하고, 저장한 메모리의 위치를 가리킨다.
```
### 문자열 연산
```
문자열 관련 연산 
    연결 : + (str + str)  ex) "a" + "b" -> "ab"
    곱 : * (str * int)  ex) "a" * 3 -> "aaa"

len() : 문자열 길이
```
### 인덱싱
```
인덱싱
    복합 형태의 자료형에 [] 대괄호를 사용하여 원하는 위치(인덱스) 번호를 넣어 값을 알아내는 방법
    *문자열은 문자가 순서대로 배치된 복합체이다. -> 인덱싱 사용 가능
    ex) a = "Hello"  a[0] -> "H", a[2] -> "l"     문자열의 인덱싱의 결과는 문자열
```
### 슬라이싱
```
슬라이싱
    하나의 요소만 뽑는 것이 아니라 범위 값을 이용해서 한 개 이상의 요소를 추출하는 방법
    형태 : 문자열[시작인덱스 : 종료인덱스]  종료인덱스는 포함 X
    and  a[0:4:2] 처럼 간격을 가지고 슬라이싱 가능    문자열의 슬라이싱의 결과는 문자열
```
**** 인덱싱 / 슬라이싱은 기존의 문자열을 분해하는 것이 아니다. 문자열은 변경 불가능한 자료형이다.
### String Method
```
문자열 관련 메소드
    메소드 : 자료형이 자체적으로 갖는 함수  <-> 함수 : 특정 자료형에 국한되지 않는 함수
        문자형 객체 뒤에 '.' 을 붙여서 문자열형만 호출할 수 있는 함수
    count() : 문자 개수 세기
    find() : 문자열 위치 찾기, 찾지 못하면 -1 반환
    index() : 문자열 위치 찾기, 찾지 못하면 오류 발생
    join() : 문자열 삽입
    upper() : 대문자로 바꾸기 <-> lower() : 소문자로 바꾸기
    strip() / lstrip() / rstrip() : 문자 공백 지우기
    replace() : 문자열 바꾸기 -> ("바꿀 문자", "새로운 요소")
    split() : 문자열 나누기
```
