## 리눅스 명령어 정리
### 파일 생성 할 때
```
touch 파일명 - 파일생성
echo "내용" >> 파일명.확장자명 --> 파일생성
```
### 폴더 이동할 떄 
```
cd 폴더명 - 이동하려는 폴더로 이동
cd .. - 상위 폴더로 이동
```
### 파일 확인, 삭제
```
ls - 선택한 폴더안 파일 확인
(ls -a : 숨겨진 파일 확인)

cat 파일이름.파일형태 - 파일 내용 출력 (텍스트 파일 내용 보기)

rm 파일명 - 파일삭제
(rm -rf 파일명 : 파일 강제 삭제)

* - all (ex rm *.txt) -- .txt 로된 파일 모두 삭제

clear - 커널화면 정리
```
