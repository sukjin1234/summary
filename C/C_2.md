### 연산자
```
수식 : 피연산자와 연산자가 결합한 형태
123 + 8 = 131   -> 123,8 : 피연산자  +,= : 연산자

산술 연산자
    +, -, *, /, %    % 연산자는 정수형에서만 사용 가능 (실수형 X)

대입 연산자 
    = : 오른쪽에 있는 값을 왼쪽에 copy 해서 저장

복합 대입 연산자(단축연산자)
    ex) a = a + 2 -> a += 2   산술연산자 모두 가능

증감 연산자
    ++ : 1 증가시킬 때 사용
    -- : 1 감소  ''

전위형 증감 연산자 : 변수 앞에 선언 -> 제시한 값에 먼저 증감 연산 수행 후 결과값을 변수에 대입
후위형 증감 연산자 : 변수 뒤에 선언 -> 제시한 값을 먼저 변수에 대입 후 연산 수행
    전위형 ++ a ++ 후위형

관계 연산자 
    연산자 기준으로 왼쪽과 오른쪽에 잇는 두 개의 피연산자를 비교하여 True 와 False로 결과를 표현
    ==, !=, >, <, >=, <=

논리 연산자
    연산식에서 2개 또는 여러 개의 피연산자를 대상으로 AND(&&), OR(||), NOT(!)의 논리 연산을 True와 False로 결과값을 출력
    AND(&&), OR(||), NOT(!)

조건 연산자
    변수명 = 조건식 ? 식1 : 식2   if True 식1을 변수에 대입
                                  if False 식2를 변수에 대입
***************
주소 참조 연산자 : &, *
    & : 해당 변수가 위치한 메모리 주소를 구함
    * : 포인터 변수가 가지고 있는 메모리 주소에서 값을 읽어옴
```

### 반복문 
```

                                            증감식;                 증감식;

```
### 분기문 
```
    프로그램을 수행할 때 지정된 위치로 이동시키는 제어문
    continue : continue문 다음에 나오는 명령은 모두 무시한채 반복문 처음으로 이동
    break : 반복문이나 switch문을 빠져나올 때 사용
```
