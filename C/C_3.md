### 함수
```
특정 작업을 함수로 처리하여 반복되는 소스코드를 줄임
구조적 프로그램을 모듈화하여 프로그램 작성과 수정이 편리하고 손쉬운 재사용
다른 사용자에게 라이브러리 제공 가능

수학 함수
    수학적인 연산을 수행하기 위한 라이브러리 함수(math.h 헤더파일 불러야함)
    ceil() : 소수점 이하 값을 무조건 올림↑
    floor() : 소수점 이하 무조건 버림

사용자 함수
    함수자료형 함수이름 (매개변수){    ex) int add (int a, int b){

    }                                       {

순환 함수 
    자기 자신을 계획해서 호출하는 함수
    탈출 조건을 반드시 선언해야함

    ex)  int fact(int n){
        if(n<=1){
            return 1;
        } else {
            return (n*fact(n-1))
        }
    }
```
### 함수 호출 방법
```
함수이름(인수) ex) add(A,B) -> 인수와 매개변수를 일대일 대응

함수 선언 -> 함수 호출 -> 함수 정의

if 사용자 함수 매개변수가 포인터 변수이면 인수는 주소 참조 연산자(&)를 붙여야함
    void func(*A)

    int main(){
        func(&a)
    }
```
### 간접 참조 연산자(*)   
```
포인터 변수와 주소 참조 연산자        *** 포인터 변수는 가르키는 주소의 값을 건드림
        int *a   //주소를 저장할 수 있는 포인터 변수 선언
        int b = 100  //정수형 변수 선언과 초깃값 대입
        a = &b     // 변수 a에 변수 b값이 저장되어 있는 주소값 대입
        *a = 200   // 변수 a가 가르키는 주소(메모리 주소)의 실제 값을 200으로 변경
```
